# JavaScript
JavaScript is a popular programming language used to make websites interactive.JS invented by Brendan Eich in 1995 and became an ECMA standard in 1997. It runs in web browsers and lets developers add features like animations, buttons, and form validation. 
JavaScript works with HTML and CSS to build modern web pages and web applications. 
JS can update & change both HTML and CSS, It can calculate, manipulate and validate data.
Installations - Visual studio code => Node.js => Local C => Settings => User variables and System variables
JS is platform independent,It is widely used and supported by all major browsers like Linux, windows, MAC and other.
